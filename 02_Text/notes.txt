stdio.h
  Стандартен (std) header с функции за in/out (IO) операции (терминал, файлове)
  Системни хедъри се #include-ват с <> (вместо "")
    #include <stdio.h>
  Функциите, които познаваме до момента:
    printf("", ...)  - форматиран output
    getchar()        - взима един (следващия) символ от stdin
    putchar('a')     - извежда/принтира подадения (един) символ на stdout


Цикъл/loop
  За многократно изпълнение на един и същи код
  while (условие)
  {
    // body/тяло
  }
  Условието се проверява преди всяко изпълнение (включително първото),
    продължаваме след тялото на цикъла, когато условието не е вярно.


char
  Реално char е целочислен тип, както int
  За да получим текст, числата са съотнесени към конкретни символи
  Най-простата система е ASCII, която включва само английската азбука
    https://www.asciitable.com
    'A' == 65, 'B' == 66 ... 'a' == 97 ... 'a' + 1 == 'b' ... '0' == 48 ...


След прочитането на цял файл
  (или при край на пренасочен input / Ctrl + D)
  Получаваме сигнал EOF (end-of-file), например от getchar
  Това също не е някаква магия, а число (константа) (-1), дефинирана в stdio


Дефиниране на функция:
  (result/output)           (input)
  "return-type"  name       "parameters"
  char           uppercase  (char param_name)
  {
    // function body (тяло, "имплементация")
    return X;
  }
